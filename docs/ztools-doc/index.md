# ZTools 开发者文档

## ZTools 一个强大、可扩展的插件系统。

使用自定义插件提升您的生产力。简单的配置，丰富的 API，以及跨平台支持。

- [开始使用](./getting-started.md)
- [API 参考](./plugin-api.md)

## 简单配置

通过标准的 plugin.json 文件轻松定义插件。无需复杂的设置。

## 丰富的 API

通过全局 ztools 对象访问系统能力，包括通知、模拟输入和持久化存储。

## 灵活的指令

使用文本、正则或全局钩子触发您的插件，以适应任何工作流。

## 跨平台

一次构建，在 Windows、macOS 和 Linux 上运行。在所有设备上获得一致的体验。
