这次优化，你最在意哪一点？
A

单一解析 API

所有入口共用 detect / resolve，消灭分叉逻辑

B

单一渲染组件

业务侧只写 <AppIcon :src />，不再手写 v-if 树

C

规范存储协议

存库/配置时统一格式（provider: / store: / file: …）

D

A+B 一起，协议最小改

解析+渲染收敛；存量字符串尽量兼容，少迁数据
